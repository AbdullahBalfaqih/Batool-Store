# **App Name**: Genci Digital Presence

## Core Features:

- Service Showcase: Display Genci's services (UI/UX design, Branding, Web Development, Marketing).
- Portfolio Display: Showcase Genci's work through an image gallery.
- Contact Form: Allow potential clients to submit inquiries.
- AI Style Suggestion: Suggest the optimal visual style guide, based on user inputs, as a lead-in to engaging Genci services for website building and other work. An AI tool will guide style creation decisions based on information in the user's requests and inputs.

## Style Guidelines:

- Primary color: Gradient from dark blue (#003049) to light blue (#40A2D8) for a modern, digital feel, evocative of clear skies or deep waters.
- Background color: Very dark desaturated blue (#0A192F) for a sleek, professional look.
- Accent color: Teal (#0093AB) for highlights and interactive elements.
- Body and headline font: 'Inter' sans-serif for a clean, readable, modern appearance.
- Use simple, line-based icons for services and key features.
- Full-screen layouts with clear sections for services, portfolio, and contact information.
- Subtle scroll animations and hover effects to add interactivity and visual interest.
